# 画像素材について

フォルダ・ファイル名はすべて半角英数字にしてあります（日本語パスだと環境によって
画像が読み込めないことがあったため、`images/cards/reimu.png` のような形に統一しました）。

## 反映済み
- カード裏面、汎用スペル／カウンタースペル／エリアの共通イラスト
- キャラクター基本形：11人分すべて
- キャラクター覚醒後（EX）：11人分すべて

## カード裏面にフォールバックしているもの
- エリアカード11種の専用イラスト：個別イラストは未提供のため、共通の area_generic.jpg が表示されます

追加のイラストを用意された場合は、`assets.js` の `CHARACTER_ART_MAP` / `AREA_ART_MAP` に
`"カード名": CARD_ART_DIR + "ファイル名.jpg"` の形で1行追加すれば反映されます。
ファイル名は半角英数字にしておくことをおすすめします。
